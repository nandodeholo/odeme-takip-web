# 💳 Ödeme Takip Uygulaması

Modern ve kullanıcı dostu bir ödeme takip uygulaması.

## 🚀 Özellikler

- ✅ Kredi kartı/banka kartı yönetimi
- ✅ Aylık dönem takibi
- ✅ Ödeme durumu işaretleme
- ✅ Otomatik sıralama (ödenmeyen üstte)
- ✅ Dönemleri genişlet/daralt
- ✅ Binlik ayraçlı para gösterimi
- ✅ Yedekleme ve geri yükleme
- ✅ Responsive tasarım (mobil uyumlu)
- ✅ Offline çalışır (localStorage)

## 📱 Kurulum

1. Web adresini ziyaret edin
2. Safari'de: Paylaş → **Ana Ekrana Ekle**
3. Uygulama gibi kullanın!

## 💾 Veriler

- Verileriniz sadece tarayıcınızda saklanır
- Hiçbir sunucuya gönderilmez
- Güvenli ve özel

## 🛠️ Teknoloji

- Pure HTML/CSS/JavaScript
- localStorage API
- Responsive Design
- PWA Ready
